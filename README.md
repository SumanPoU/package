## Local development

```bash
pnpm install
pnpm dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Packages

```bash
pnpm add @itzsa/table
pnpm add @itzsa/nepali-input
pnpm add @itzsa/editor @itzsa/nepali-input
```

## Build

```bash
pnpm run build:packages
pnpm run build
```
